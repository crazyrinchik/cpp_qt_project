1. load googletest in terminal by this command: git clone https://github.com/google/googletest.git -b v1.14.0
2. go to this loaded file googletest in terminal by command cd {path to file} 
3. create file build: mkdir build
4. go to this file: cd build
5. make build: cmake..
6. finish build: make
7. copy and paste the correct absolute path in pro files.
